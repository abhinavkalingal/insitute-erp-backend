# AI Development Rules

- This is a production SaaS ERP.
- Never generate MVP code.
- Follow Clean Architecture.
- Follow SOLID principles.
- Use NestJS + PostgreSQL + Prisma.
- Always implement validation.
- Always implement authorization.
- Always implement audit logs.
- Never duplicate code.
- Every API must have Swagger documentation.
- Every business table must include instituteId.
- Use soft deletes.
- Never expose internal errors.
- Follow the existing folder structure.
- Wait for approval before creating the next module.
